# 🧾 POS Server - Point of Sale Dashboard

Ένα πλήρες σύστημα διαχείρισης πωλήσεων (POS) με backend server (Node.js/Express) και frontend dashboard. Το σύστημα επιτρέπει τη διαχείριση προϊόντων, αποθέματος, πωλήσεων και εκτύπωση αποδείξεων.


## ✨ Χαρακτηριστικά

### Backend (Node.js Server)
- ✅ REST API για προϊόντα και πωλήσεις
- ✅ Αυτόματη ενημέρωση αποθέματος
- ✅ Αποθήκευση ιστορικού πωλήσεων
- ✅ Υπολογισμός ΦΠΑ 24%
- ✅ CORS enabled

### Frontend Dashboard
- ✅ **Πίνακας Αποθέματος** - Οπτικοποίηση stock με χρωματικές ενδείξεις
- ✅ **Κατάλογος Προϊόντων** - Emojis, τιμές, διαθέσιμο απόθεμα
- ✅ **Καλάθι Αγορών** - Προσθήκη/αφαίρεση προϊόντων, δυναμική ενημέρωση
- ✅ **Αυτόματοι Υπολογισμοί** - Καθαρή αξία, ΦΠΑ, τελικό σύνολο
- ✅ **Εκτύπωση Απόδειξης** - Επαγγελματική απόδειξη με αναλυτικά στοιχεία
- ✅ **Ιστορικό Συναλλαγών** - Πλήρες αρχείο πωλήσεων
- ✅ **Σύνδεση με Server** - Real-time ενημέρωση stock
- ✅ **Responsive Design** - Λειτουργεί σε όλες τις συσκευές
- ✅ **Offline Indicator** - Οπτική ένδειξη κατάστασης σύνδεσης

## 🛠 Τεχνολογίες

### Backend
- **Node.js** - JavaScript runtime
- **Express.js** - Web framework
- **CORS** - Cross-origin resource sharing
- **Body-parser** - JSON parsing
- **File System (fs)** - Αποθήκευση δεδομένων σε JSON αρχεία

### Frontend
- **HTML5** - Δομή σελίδας
- **CSS3** - Στυλ, grid layout, animations
- **JavaScript (Vanilla)** - Χωρίς εξωτερικές βιβλιοθήκες
- **Fetch API** - Επικοινωνία με backend
